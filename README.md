# **Pekeninhos**
## Projeto de Desenvolvimento de Sistemas II

Projeto desenvolvido com objetivo de criar uma loja para uma marca ficticia a **Pekeninhos**,
que como proposito inicial tem:

 ### Area do cliente 
    - Login com email/nome de usuario, senha e uma passkey por email.
    - Carrinho para adição de produtos.
    - Lista de Desejos para listar produtos.
    - Historico de Compras.
    - Pagina Principal com Produtos. 
    - Rodapé com informações de contato a empresa. 

### Area da Gestao 
    - Login com credenciais da Pekeninhos.
    - Criação, Visualização, Edição e Exclusão de produtos.
    - Alterar Perfis.
