# **Pekeninhos**
## Projeto de Desenvolvimento de Sistemas II

Projeto desenvolvido com objetivo de criar uma loja para uma marca ficticia a **Pekeninhos**,
que como proposito inicial as seguintes areas:

 ### Area do cliente 
    - Pagina Principal com Produtos. 
    - Login com email/usuário, senha e uma passkey por email.
    - Cadastro com nome, email, CPF, telefone, senha e botton.

#### Barra de ações
    - Login e cadastro 
    - Pesquisa de produtos
    - Inseerção de CEP
    - Carrinho para adição de produtos.

#### Barra de links
    - Opções como: brinquedos, acessórios, roupas.

#### Produtos
    - Ver especificações do produto.
    - Lista de Desejos para listar produtos.
    - Historico de Compras. 

---

### Area da Gestao 
    - Login com credenciais da Pekeninhos.

#### Usuários 
    - Visualizar histórico de compras, perfis dos usuários.

#### Produtos
    - Criação, Visualização, Edição e Exclusão de produtos.
